## 🚀 Projects

### 📊 Depression ML Data Analysis
Machine learning pipeline built on a **100k-entry student lifestyle dataset** to identify potential depression risk.

**Highlights**
- Built a **binary classification pipeline** focused on recall and F1-score optimization
- Developed custom features such as **Burnout Index** and **Social Escape Ratio**
- Implemented a **Calibrated Random Forest with Isotonic Regression**
- Improved recall **21x** by optimizing the prediction threshold (~0.21)
- Packaged the final model into a **standalone .exe tool** for real-time risk assessment and lifestyle recommendations
